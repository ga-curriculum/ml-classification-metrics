## About
An introduction to Classification metrics and definition of the common ones.

## Learning Objectives

*You will be able to:*

- Compare regression and classification metrics.
- Define common metrics and the confusion matrix.
- Describe the ROC curve and the AUC score.

## Prerequisites
- Interpret and use Python dictionaries.
- Build `pandas` `DataFrames` from dictionaries.
- Perform simple data manipulation on `pandas` objects.
- Build `for` and `while` loops in Python.
- Use `pip install` for package management.
- Create data visualizations in a noteboook using `matplotlib` and `seaborn`.
- Build a linear regression model.
- Understand bias / variance and implment train-test split.
